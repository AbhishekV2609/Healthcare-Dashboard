# 🏥 Healthcare Dashboard (Frontend Skills Test)

## 📌 Overview

This project is a responsive Healthcare Dashboard built as part of a Frontend Development Skills Test. It displays patient data using an API and visualizes diagnosis history with charts.

The UI is implemented based on the provided Adobe XD design and focuses on clean layout, usability, and performance.

---

## 🚀 Features

* 📊 Blood Pressure Chart (Systolic & Diastolic)
* 📅 Dynamic filtering (Last 1–12 months)
* 👤 Patient profile details
* 📋 Sidebar with patient list (click to switch)
* ❤️ Health stats (Respiratory Rate, Temperature, Heart Rate)
* 🎨 Pixel-accurate UI based on Adobe XD
* ⚡ Responsive layout

---

## 🛠️ Tech Stack

* React.js
* Axios (API calls)
* Chart.js (data visualization)
* Lucide-react (icons)
* CSS (custom styling)

## ⚙️ Setup Instructions (Important)

After installing, start the project using:

```bash
npm start
```

The application will run on:
